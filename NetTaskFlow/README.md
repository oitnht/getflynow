
# Replit React App

A modern React application built with TypeScript, Vite, and Tailwind CSS, optimized for Replit.

## Features

- ⚡ Vite for fast development
- ⚛️ React 18 with TypeScript
- 🎨 Tailwind CSS for styling
- 🔧 ESLint for code quality
- 📱 Responsive design ready

## Getting Started

1. Click the "Run" button to start the development server
2. Your app will be available at the preview URL
3. Edit `src/App.tsx` to start building your application

## Project Structure

```
src/
├── components/     # Reusable UI components
├── App.tsx        # Main application component
├── main.tsx       # Application entry point
└── index.css      # Global styles with Tailwind

## Development

- **Run**: Use the Run button or `npm run dev`
- **Build**: `npm run build`
- **Deploy**: Use Replit Deployments

## Adding Dependencies

Use the Shell to install packages:
```bash
npm install package-name
```

Happy coding! 🚀
